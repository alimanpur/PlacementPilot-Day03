# Placement Tracker

Part of the **30 Days 30 Projects** challenge.

## Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + Prisma + PostgreSQL

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
